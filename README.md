# Data Cleaning and Preparation | SkillCraft Technology Internship - Task 2

## Project Overview

This project was completed as part of the Data Analytics Internship at SkillCraft Technology. The objective of this task was to clean and prepare raw sales data using Python and the Pandas library in Google Colab.

The project involved handling missing values, checking duplicate records, converting data types, and exporting the cleaned dataset for further analysis.

---

## Objectives

* Load dataset into Python environment
* Perform data cleaning and preparation
* Handle missing values
* Detect and remove duplicate rows
* Convert date columns into proper datetime format
* Export cleaned dataset into a new CSV file

---

## Features Implemented

* Dataset Loading using Pandas
* Missing Value Detection
* Duplicate Row Checking
* Data Type Conversion
* Date Formatting
* Cleaned Dataset Export
